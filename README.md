  纯整理发票报销烦了自己搞的玩的，AI生成的项目

一、整体技术架构
这是一个纯前端单文件离线应用，不需要任何后端服务器，所有 PDF 解析、文本提取、重命名、ZIP 打包操作都在用户本地浏览器完成，数据不会上传到任何外部服务器，绝对安全。
二、核心技术栈
1. 基础技术（无任何外部依赖）
技术：
  HTML5	页面结构和语义化标签
  原生 CSS3	所有界面样式（卡片、表单、表格、按钮等），完全手写，不依赖任何 CSS 框架
  原生 JavaScript ES6+	所有业务逻辑（文件处理、PDF 解析、正则匹配、文件名生成）
2. 第三方库（仅 4 个，均为前端行业标准库）
所有库都通过 CDN 引入，体积小、加载快、稳定性高：
库名：
  1.PDF.js	3.4.120
  2.JSZip	3.10.1
  3.FileSaver.js
  4.SortableJS
